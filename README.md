# 桂林旅游学院调查问卷数据集

## 📋 项目简介

本数据集由**桂林旅游学院数据学院团委**收集整理,是一份经过脱敏处理的调查问卷数据,现以开源形式发布,供学术研究、数据分析学习等用途使用。

## 📊 数据文件

- `附件1.xlsx` - 调查问卷原始数据(已脱敏)

## 🔒 隐私保护

本数据集已进行严格的脱敏处理:
- 已移除所有个人身份识别信息(PII)
- 所有敏感信息已被匿名化处理
- 符合数据保护和隐私法规要求
